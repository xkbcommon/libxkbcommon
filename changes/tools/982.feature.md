`xkbcli {dump-keymap,interactive*}`: Added `--strict` for parsing
keymaps in *strict* mode. This useful mainly for debugging.

`xkbcli {compile-keymap}`: Added `input-strict` and `--strict` for *parsing*
keymaps in *strict* mode. `--strict` additionally enables *serializing*
in strict mode. This useful mainly for debugging.
